# Unearthed_Robot_Pres

**Team:** Radioactive Brix

## Robot Configuration

- Left wheel: Port C
- Right wheel: Port D
- Wheel diameter: 56mm
- Axle track: 80mm

## Quick Start

1. **Add your first mission:**
   ```bash
   python ../new_mission.py
   ```

2. **Upload to hub:**
   - Upload ALL .py files to your SPIKE Prime hub

3. **Run the menu:**
   - Run season_menu.py on your hub
   - Use LEFT/RIGHT buttons to select mission
   - Press CENTER to run
